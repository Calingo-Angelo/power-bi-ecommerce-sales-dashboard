# Dataset

This project uses the **Brazilian E-Commerce Public Dataset by Olist**, available through Kaggle.

## Source

[Olist Brazilian E-Commerce Public Dataset](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)

## Included Data

The source dataset contains information about:

- Customers
- Orders
- Order items
- Products
- Sellers
- Payments
- Customer reviews
- Product-category translations
- Geolocation data

The raw CSV files are not included in this repository because the dataset is publicly available and some files are large. The original data can be downloaded directly from Kaggle using the link above.
